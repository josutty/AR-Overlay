Place your compiled marker.mind file here.

How to generate it:
1. Go to: https://hiukim.github.io/mind-ar-js-doc/tools/compile
2. Upload your marker image (e.g. hiro.png or any high-contrast image)
3. Click "Start" to compile
4. Download the .mind file
5. Rename it to marker.mind and place it in this folder

The app expects: public/targets/marker.mind
